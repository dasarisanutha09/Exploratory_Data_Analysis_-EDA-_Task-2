Exploratory Data Analysis (EDA) using Python
Project Description

This project performs Exploratory Data Analysis (EDA) on a dataset to understand the structure, distribution, and important patterns present in the data.

The main objective of this project is to clean the dataset, analyze statistical information, identify relationships between variables, and discover meaningful insights using Python.

Analysis Performed
1. Data Loading and Inspection

The dataset was imported into Python using Pandas and the initial structure of the data was examined.

2. Dataset Information Analysis

The number of rows, columns, and data types were checked to understand the dataset structure.

3. Missing Value Analysis

The dataset was checked for missing or empty values to improve data quality.

4. Statistical Summary

Important statistical measures were calculated:

Mean
Minimum Value
Maximum Value
Count
5. Category Analysis

Different categories were analyzed to identify patterns and compare data values.

6. Correlation Analysis

Correlation analysis was performed to understand relationships between numerical variables.

Different graphs were created to understand:

Data distribution

Relationship between variables
